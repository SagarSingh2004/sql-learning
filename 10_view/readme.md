# SQL Topic: Views

## Database & Tables
- **Database:** PostgreSQL
- **Tables used:** employees, departments

## Description
This file contains queries to create and manage **views** in PostgreSQL. Views allow you to:

- Simplify complex queries by storing them as virtual tables.
- Present specific columns or filtered data to users.
- Aggregate data or perform calculations on-the-fly.
- Implement security by hiding sensitive information like salaries.
